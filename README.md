드라마의 첫 회 시청률과 흥행, 그리고 배우들의 시청률 스탯을 탐색해보자
=====

드라마 한편이 흥행하면 배우, OST, 등장인물 심지어 촬영 장소까지 흥행하게 된다. 우리는 드라마 한편이 가져오는 경제적, 문화적 파급력은 생활에서 피부로 느낀다. 그렇다면 드라마의 흥행에는 어떤 요소가 영향을 미칠까? 드라마를 전혀 모르는 작성자가 오로지 데이터만으로 드라마를 들여다 본 스토리.
